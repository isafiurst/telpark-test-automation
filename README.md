1. Clonar o repositório:
git clone <URL>

2. Entrar na pasta:
cd .\telpark-test-automation\

3. Instalar o Cypress:
npm install -D cypress

4. Instalar a estruta de pastas:
./node_modules/.bin/cypress open

5. Instalar o xpath:
npm install -D cypress-xpath

6. Abrir o Cypress:
npm run open

7. Gerar relatórios:
npm run test
